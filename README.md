# agentops-powderline

Milestone-driven agent orchestration for OpenClaw. Picks GitHub issues from a milestone, plans an approach via subagent, then codes and PRs the solution.

## Structure

```
SKILL.md          — skill definition (OpenClaw skill format)
agents/           — agent workspace templates for plan/code subagents
scripts/          — deterministic helpers
references/       — docs loaded on demand
assets/           — output templates
```
